 PCRehberim - Bilgisayar Satın Alma Rehberi

Bu proje, bilgisayar donanımları hakkında bilgisi olmayan kullanıcıların doğru laptop veya masaüstü bilgisayarı seçmelerine yardımcı olmak amacıyla geliştirilmiş bir web platformudur.

 Proje Hakkında
Kullanıcılar bütçelerine ve ihtiyaçlarına (oyun, iş, okul) göre en doğru bilgisayarı seçmek için rehber içeriklere ulaşabilirler.

**Kullanılan Teknolojiler:**
* WordPress (CMS)
* Elementor (Sayfa Tasarımı ve UI)
* HTML/CSS (Özel stil düzenlemeleri için)
